# GRC-App
Demo Application for GRC Remediation Engine https://github.com/devjoshi2005/Grc-Compliance-Engine


# Usability Guide
1) Clone this repository in your terminal via `git clone https://github.com/devjoshi2005/GRC-App`
2) After cloning, open your respective folder where you cloned,next perform this command `source venv/bin/activate`
3) After activating virtual python environment, run this application using `python3 api/app.py` in your terminal
