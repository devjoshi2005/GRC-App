Demo Static Application for GRC Remediation Engine https://github.com/devjoshi2005/Grc-Compliance-Engine

Note that: you have to use aws iam credentials and azure service principal credentials with required least permissions for scan execution.The whole pipeline scan process may take 5-10 minutes depending upon resources,regions you chose

# Usability Guide
1) Clone this repository in your terminal via `git clone https://github.com/devjoshi2005/GRC-App`
2) execute this command in terminal to create and use python virtual environment `python -m venv venv && source venv/bin/activate && pip install -r requirements.txt` with required modules
3) After activating virtual python environment, run this application using `python3 api/app.py` in your terminal
